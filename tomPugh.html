  • Radiation

    Manipulation
    - Delete original record (marked as 0 by Met Office) if the modified record (Marked as 1) is available
    - If data miss on some day/hour, insert the time and set the record as NAN
    - Only retrieve 24 hours sum record, ignore hourly record.
    - Listed out missing day/hour (This is listed in the last column, I recorded it as the number of record in the dataset)
    - Separate dataset by stations (Not applied here as we got only 1 station)
    - Listed out which stations are working on which days? (Not applied here as we got only 1 station)

    Explanation
    - The dataset is based on station Shawbury, from 2001 to 2017
    - ob_end_time is date, Data before 2001 is filled by climatology based on 643
    - Radiation_Kj_m2 is the daily gross glbl_irad_amt, further info available here
    - src_id is station number, only 1 station
    - Number_of_records is the number of records which are used to calculate the mean value

    Data
    - Link (need to login uni account): Radiation_file_pack; Bifor_Radiation.csv

  • Temperature_Mean

    The data is prepared for Dr Tom Pugh.

    Manipulation
    - Calculate mean temperature from hourly record.
    - Delete original record (marked as 0 by Met Office) if the modified record (Marked as 1) is available
    - use station 643 records primarily, then fill NaN
    - set any day with records less than 5 as NaN, grab the data from 622 (only 10 days, do that manually)

    Explanation
    - The dataset is based on station Shawbury
